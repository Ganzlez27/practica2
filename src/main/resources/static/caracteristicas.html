Características de Godot

Compatibilidad de plataformas

Editor y proyectos exportados

  1. Windows (x86 y ARM, 64-bit y 32-bit)
  2. macOS (solo ARM x86 y 64 bits)
  3. Linux (x86 y ARM, 64 y 32 bits)
  4. Android (editor experimental)
  5. Navegadores web (experimental en 4.0, se recomienda Godot 3.x con HTML5)

Notas técnicas

  • Binarios vinculados estáticamente: se ejecutan en cualquier distribución si se compilan sobre una base antigua.
  • Los binarios oficiales se compilan con Godot Engine buildroot para compatibilidad con múltiples distribuciones Linux.

Proyectos exportados

  1. iOS
  2. Consolas

Renderizado

Forward+

El renderizador más avanzado, adecuado solo para plataformas de escritorio. Se usa por defecto en dichas plataformas. Este renderizador utiliza Vulkan, Direct3D 12 o Metal como controlador de renderizado, y usa el backend RenderingDevice.

Mobile

Con menos funciones, pero renderiza escenas simples más rápido. Adecuado tanto para plataformas móviles como de escritorio. Se usa por defecto en plataformas móviles. Utiliza Vulkan, Direct3D 12 o Metal como controlador de renderizado, y el backend RenderingDevice.

Compatibility

También llamado Compatibilidad GL. Es el renderizador menos avanzado, ideal para plataformas móviles y de escritorio de bajos recursos. Se usa por defecto en la plataforma web. Utiliza OpenGL como controlador de renderizado.
